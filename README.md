# Iris-Dataset-Analysis

## Overview
This project predicts iris flower species using machine learning and using the iris.csv dataset.
- Original Dataset: UCI Machine Learning Repository  
- Creator: R.A. Fisher (1936)  
- Source (Kaggle): https://www.kaggle.com/datasets/uciml/iris 

## Dataset
- 150 samples
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Species

## What we did
- Loaded and explored dataset
- Removed unnecessary column (Id)
- Encoded categorical data
- Created new features:
  - Sepal Area
  - Petal Area
  - Ratios
- Split data into training and testing
- Applied feature scaling
- Trained 3 models:
  - Decision Tree
  - Neural Network
  - Logistic Regression
- Compared performance using:
  - Accuracy
  - Precision
  - Recall
  - F1-score
